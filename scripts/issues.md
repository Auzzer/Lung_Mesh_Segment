## Reason

- Stage 1: the loss plunges from $\sim 4.3 \mathrm{e} 3$ to $\sim 1.0$ while $\theta$ comesto $\sim 20.3$, and ${\alpha}$ explodes to $\sim 6.5 \times 10^8$.
- Stage 2: gradients for $\delta$ are essentially zero and the optimizer stops immediately.

What's happening:
1. The data loss is
$\mathcal{L}_{\text {data }}=\left\|u^{\star}-u_{\text {obs }}\right\|^2 /\left\|u_{\text {obs }}\right\|^2 .$
If the material is extremely stiff ( $\alpha \rightarrow \infty$ ), the solution under gravity tends to zero ( $u^{\star} \rightarrow 0$ ), so $\mathcal{L}_{\text {data }} \rightarrow\left\|u_{\text {obs }}\right\|^2 /\left\|u_{\text {obs }}\right\|^2=1$.
This is a degenerate but locally optimal sol.

2. In that stiff limit, both the forward state and adjoint satisfy$
u^{\star} \sim K^{-1} b \propto \alpha^{-1} \text { and } \lambda \sim K^{-1} \mathrm{RHS} \propto \alpha^{-1} .$
Hence the element sensitivity scales like$
q_k \propto\left(r \cdot u^{\star}\right)(r \cdot \lambda) \propto \alpha^{-2}$. 
the Stage-2 chain rule multiplies by $\alpha_k$, so
$
\nabla_{\delta_k} L \sim \alpha \cdot \alpha^{-2}=\alpha^{-1} \rightarrow 0$

3. Regularization is currently ineffective at the stiff optimum. Because $\nabla u^{\star} \approx 0$ when theta is huge, the TV contributes neither loss nor gradient.
